Add-on to Advanced Custom fields (ACF) for showing multiple ACF fields in a single row on front-end. Great for short text fields or having 2x date fields next to each other. Once activated, 'Row' field will be available under the 'Edit Field Group' section in ACF field types.

== Description ==

<h3>Display up to 4 Advanced Custom Fields (ACF) fields in a single row on the front-end</h3>

Advanced Custom Fields: Multiple Fields per row, is an add-on to Advanced Custom Fields for showing multiple ACF fields in a single row. You can show up to four fields in a row. It is very helpful for managing form display layout as you desire. Especially handy with 2x short fields or date fields next to each other.

__Note__: Works up to ACF 4.4


= Usage =
1. Once activated, 'Row' field will now be available under the 'Layout' section in ACF field types
1. Add 'Row' field in your field setting. Choose 'Row Open' option in 'Type' to start row
1. Choose the number of columns you would want
1. Add your fields below the 'Row Open' option
1. To end row, add 'Row' field. But this time choose 'Row Close' option in 'Type'

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' Plugin Dashboard
1. Click on 'Upload Plugin'
1. Select 'advanced-custom-fields-row-field.zip' from your computer
1. Install Now
1. Activate the plugin on the WordPress Plugin Dashboard
1. 'Row' field will now be available under 'Layout' section in ACF field types

= Using FTP =

1. Extract 'advanced-custom-fields-row-field.zip' to your computer
1. Upload the 'advanced-custom-fields-row-field' directory to your '/wp-content/plugins/' directory
1. Activate the plugin on the WordPress Plugins dashboard
1. 'Row' field will be added under 'Layout' section in ACF field types
